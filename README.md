# memoriesfond
Memories Fond | Heartwarming Children's Books by Andrew Abel

## Run locally

```bash
python3 -m http.server 8080 --directory /home/aabel/github/memoriesfond & sleep 0.5 && xdg-open http://localhost:8080
```

Then open [http://localhost:8080](http://localhost:8080) in your browser. Press `Ctrl+C` to stop the server.
